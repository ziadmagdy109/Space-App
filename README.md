# 🚀 Space Explorer

A modern Flutter application that explores the beauty of our solar system with a clean and interactive user interface.

The app provides an engaging experience for discovering planets through smooth animations, swipe navigation, and detailed information screens.

# 🚀 Getting Started

### Clone the Repository

```bash
git clone [https://github.com/ziadmagdy109/SpaceExplorer.git](https://github.com/ziadmagdy109/Space-App)
```


### Install Dependencies

```bash
flutter pub get
```

### Run the App

```bash
flutter run
```
